javascript中的正则表达学习

定义

###在javascript中我们可以通过内建的类定义一个正则表达式

var re = new RegExp(‘[0-9]’);

###实际上RegExp类可以接收两个参数，除了本身的需要匹配的模式字符串外，还可以定义额外处理方式的第二个参数

var re = new RegExp(‘[0-9]’,’g’);

###我们可以打印看下re的值

console.log(re)// /[0-9]/g

###所以我们可以用第二种方式定义正则表达式

var re = /[0-9]/g

###如果我们需要匹配’\’

####记住，一个完整的正则表达式里’\’后面一定会有会另外一个字符

#javascript中的正则表达式
其实上面我们已经简单的介绍了正则表达式的定义，但是要如何的使用呢？在javascript中RegRxp和String中都有方式处理这些正则表达式

  • test – RegRxp中的test方法用来检查字符串是否匹配正则表达式，返回布尔值
  • exec – RegRxp中的exec方法返回包含第一个匹配的的数组或null；
  • match – String中的match方法返回包含所有匹配子字符串的数组
  • replace – String中的replace方法用来替换匹配的字符串，支持正则匹配
  • split – String中的split匹配指定符号来分割字符串，支持正则匹配，返回数组

一个实例对象除了方法当然还有属性，一个正则表达式有以下属性：

  • global – 布尔值，若全局选项g已设置则返回true，否则返回false；
  • ignoreCase – 布尔值，若忽略大小写选项i已设置则返回true，否则返回false；

#元字符

在正则表达式中有一些特殊字符是不能直接使用的，必须对它进行转义后才可以，比如’\’，因为这些字符在正则表达式中有特殊的语法含义，这类字符被称为元字符，正则表达式中的元字符有

. \ / ? * + { } [ ( ) ] ^ $ |

#取非匹配

####很多编程语言中都使用“!”取非操作，包括javascript。正则表达式中也有取非操作，比如/[^0-9]/就是一个取非操作的正则表达式了。

特殊字符

在正则表达式中有一些特定的字符集合可以用特殊字符来替代，这些特殊的字符并不是必不可少的，但是却可以给我们带来不少方便。/[0-9]/就完全可以写成这样

常见的特殊字符有

  • /\d/ 任何一个数字字符，等价于/[0-9]/
  • /\D/ 任何一个非数字字符，等价于/[^0-9]/
  • /\w/ 任何一个数字字母下划线，等价于/[a-zA-Z0-9_]/
  • /\W/ 任何一个非数字字母下划线，等价于/[^a-zA-Z0-9_]/
  • /\s/ 任何一个空白字符，包括换页符、换行符、回车符、制表符和垂直制表符，等价于[\f\n\r\t\v]
  • /\S/ 任何一个非空白字符，等价于[^\f\n\r\t\v]
  • /./ 任何一个非空格回车，等价于[^\n\r]
    相同字母大小写总是进行取非操作的

#贪婪模式(重复模式)
以匹配一个邮箱username@email.com为例，邮箱包括一个用户名，@符号与一个域名，用户名与域名的的个数无法判断，但是可以肯定的是用户名与域名至少有一个字符

贪婪模式有几个符号我们需要了解

1、 + 表示出现一次或多次，至少一次
2、 * 表示出现任意次数，零次，一次，或多次
3、 ? 表示出来零次或一次，最多一次
4、 {n} 能且只能出现n次
5、 {n,} 最少出现n次，最多不限
6、 {n,m} 最少出现n次，最多出现m次

#防止过度匹配

有这样一段HTML文本：

如果现在要将及其中间的文本匹配出来，正则表达式可以这样写：

当使用贪婪模式的时候，”.”会最大程度地进行字符匹配，所以输出了整个字符串。而在惰性模式中，”.?”只进行最小限度的匹配，所以完整的输出了我们需要的字符串。

惰性模式的语法很简单，即是在贪婪模式后面加上一个“?”即可。

1、 –> ?
2、+ –> +?
3、{n,} –> {n,}?

JavaScript常用正则表达式

在前端项目中经常会用到一个些正则表达，所以就在网上整理一些常用的正则表达式，如有写的不对的地方，可以多多交流

数字类表达式

  1. 数字 /^[0-9]*$/ 或者 /^\d$/
  2. n位数字 /^[0-9]{n}$/ 或者 /^\d{n}$/
  3. 到少n位数字 /^[0-9]{n,}$/ 或者 /^\d{n,}$/
  4. n-m位数字 /^[0-9]{n,m}$/ 或者 /^\d{n,m}$/
  5. 正负整数 /^(-|+)?\d+$/
  6. 正负小数 /^(-|+)?\d*(.\d+)$/
  7. 2位小数的正实数 /^[1-9]\d+(.\d{2})$/
  8. 大于0的正整数 /^+?[1-9]\d*$/
  9. 1-99的正整数 /^[1-9]{1,2}$/
  10. 非正整数 /^-[0-9]$/ 或者 /^-[0-9]\d$/
  11. 正负浮点数 /^(-?\d+)(.\d+)$/

校验字符串表达式

  1. 汉字 /^[\u4e00-\u9fa5]{0,}$/
  2. 大小写字母 /^[a-zA-Z]+$/
  3. 大小字字母数字 /^[a-zA-Z0-9]+$/
  4. 26位大写字母 /^[A-Z]+$/
  5. 26位小写字母 /^[a-z]+$/
  6. 字母数字下划线 /^[a-zA-Z0-9_]+$/

其实类需求表达式

  1. 11号手机号码 /^(13|14|15|18|19)[0-9]{9}$/
  2. 邮箱表达式 /^^\w+([-+.]\w+)@\w+([-.]\w+).\w+([-.]\w+)*$$/
  3. 2位小数价格 /^[0-9]+(.[0-9]{1,2})$/

mac sublime text 3 安装插件

sublime text3 安装插件管理器Package Control

打开Sublime，按下Control + (Mac)或者Ctrl +(Windows)，然后粘贴上下面的代码

import urllib.request,os; pf = ‘Package Control.sublime-package’; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), ‘wb’).write(urllib.request.urlopen( ‘http://sublime.wbond.net/‘ + pf.replace(‘ ‘,’%20’)).read())

安装好插件管理器后，按下command+shift+p,搜索Install Package,可以搜索想要安装的插件包

按下command+shift+p,搜索Remove Package,可以搜索想要移除的插件包
